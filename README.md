# Learn Express

Learn Express.

## How to Run

Start:

```sh
npm start
# or
yarn start
```

Development:

```sh
npm run dev
# or
yarn dev
```

## REST API Endpoints

- GET `/`: Hello
- GET `/users`: List of users
- POST `/users`: Create new user
- DELETE `/users`: Delete all users
- DELETE `/users/:id`: Delete one user by id
- PUT `/users/:id`: Update one user by id
